# 每日报告目录

本目录存储每日自动生成的期货市场分析报告。

## 文件命名规则

- 报告文件: `daily_report_YYYYMMDD.md`
- 数据文件: `daily_data_YYYYMMDD.json`

例如: `daily_report_20260319.md` 是2026年3月19日的报告。

## 报告内容

每日报告包含:

- 📋 市场概况
- 🎯 推荐分布统计
- 🟢 做多推荐 TOP10
- 🔴 做空推荐 TOP10
- 📈 分类行情（金属、黑色、化工、农产品、金融）
- ⚠️ 风险提示

## 自动更新

报告每天自动生成并提交到GitHub仓库。
